# 📊 Exploratory Data Analysis (EDA) - Global Superstore Dataset

## 📌 Project Overview
This project is part of the Oasis Infobyte (OIBSIP) Data Analytics Internship (Level 1).

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Global Superstore dataset to identify meaningful patterns, trends, and business insights using Python.

---

## 🎯 Objectives
- Understand the dataset structure.
- Perform data cleaning and preprocessing.
- Analyze sales, profit, discount, and quantity.
- Identify top-performing categories, sub-categories, segments, and regions.
- Visualize data using charts and graphs.
- Generate business insights for decision-making.

---

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📂 Dataset
**Dataset Name:** Global Superstore Dataset

The dataset contains information about:
- Sales
- Profit
- Discount
- Quantity
- Category
- Sub-Category
- Region
- Customer Segment
- Shipping Cost
- Order Details

---

## 📈 Analysis Performed

### Data Cleaning
- Checked missing values
- Removed duplicates
- Converted date columns
- Created additional date features

### Exploratory Data Analysis
- Sales Distribution
- Profit Distribution
- Discount Distribution
- Quantity Distribution
- Sales by Category
- Sales by Sub-Category
- Sales by Segment
- Sales by Region
- Correlation Heatmap

---

## 📊 Key Insights
- Technology generated the highest sales.
- Phones were the best-selling sub-category.
- Consumer segment contributed the maximum sales.
- Central region recorded the highest sales.
- Higher discounts negatively affected profit.
- Sales and shipping cost showed a strong positive correlation.
